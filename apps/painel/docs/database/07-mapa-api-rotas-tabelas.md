# Mapa API Rota -> Tabelas

Este documento mapeia as rotas atuais de `frontend/src/app/api` para as tabelas MySQL que elas leem e/ou escrevem no projeto atual.

## Metodologia

- O mapeamento foi gerado por analise estatica das rotas `route.ts` e de suas dependencias locais em `frontend/src`.
- Quando a rota possui `GET`, as tabelas encontradas no grafo local foram classificadas como `lidas` pela rota.
- Quando a rota possui `POST`, `PUT`, `PATCH` ou `DELETE`, as tabelas encontradas no grafo local foram classificadas como `escritas` pela rota.
- Em rotas que possuem leitura e escrita no mesmo arquivo/modulo, a mesma tabela pode aparecer nos dois conjuntos.
- Como o projeto usa repositories e composicao de servicos, este mapa e o retrato tecnico mais util do estado atual, mas nao substitui a leitura do dominio quando a rota for sensivel.

- Total de rotas mapeadas: `156`.
